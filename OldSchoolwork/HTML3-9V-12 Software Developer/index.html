Какво е Software Developer

Това са хората, които програмират,измислят
дизайна софтуерното приложение, тестват
го и проучват за целевата ти аудитория и 
други неща. Те кодират приложението 
преди да е изпратено за проверка 
нагоре по йерархията.

Някои от езиците за програмиране

  • C#
  • C++
  • C
  • Java
  • JavaScript
  • HTML
  • PHP
  • Visual Basic
  • Python
  • CSS и много други

Какво ни е нужно, за да станем
софтуерен разработчик

  1. Да знаем поне един език за програмиране
  2. Да пишем много код на ден
  3. Повече от един език за програмиране,
    защото един език не ни прави
    пълноценни софтуерни разработчици
  4. Mного практика и да търсим
    информация във
    всеки възможен източник
  5. !!!Постоянство!!!
